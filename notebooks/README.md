
### Notebooks naming
- `0.*.ipynb`Exploration: loading raw files, sanity-checking array shapes, checking missing data ratios, and understanding your feature scales.
- `1.*.ipynb` Data Processing / Feature Engineering: rototype transformations (e.g., merging ZIP-level housing data with county-level mortality data) before moving into `src/data_processing.py`.
- `2.*.ipynb` Visualization: exploratory geographic or temporal plots.
- `3.*.ipynb` Modeling: Prototyping individual models (e.g., LightGBM, dynamical model, etc..).
